# Internship Academy Backend

## Tech Stack
- Python 3.13
- FastAPI
- Uvicorn

## Modules
- Session Scheduler
- AI Mentor

## Setup

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python -m uvicorn app.main:app --reload
```
